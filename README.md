# Aerospace-Structures

Three MATLAB live scripts for the structural analysis and design of a civil aircraft fuselage. 

- Final_Fuselage_with_ESDU.mlx optimises a semi-monocoque fuselage for minimum mass against buckling, pressurisation, and shear flow constraints using Ashby material selection and discrete parameter search.
- Load_case_three.mlx computes shear force and bending moment distributions along the fuselage for a two-point landing load case.
- vn_diagram.mlx constructs a V-n diagram for manoeuvre and gust envelopes in accordance with CS-25.335, using XFOIL-derived aerodynamic data for a NACA SC(2)-0412 aerofoil corrected for finite-wing effects.
